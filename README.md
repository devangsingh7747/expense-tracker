# Expense Tracker Dashboard

A modern Expense Tracker Dashboard built using React and Tailwind CSS with charts, animations, dark/light mode, and LocalStorage persistence.

This project provides a clean and interactive interface for managing personal finances.

---

## Features

• Add Income and Expenses
• Edit Transactions
• Delete Transactions
• Auto Balance Calculation
• Expense Category Pie Chart
• Monthly Summary Bar Chart
• Animated Counters
• Smooth Chart Animations
• LocalStorage Persistence
• Responsive Dashboard UI

---

## Tech Stack

Frontend:
- React (Vite)
- TailwindCSS
- Recharts

State Management:
- React Context API

Storage:
- LocalStorage
